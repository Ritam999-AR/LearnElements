Help & Instructions

Getting Started

The 7-Day Periodic Table Challenge is designed to help you master all 118 elements through interactive learning and daily quizzes.

  • Each day unlocks new elements to learn
  • Complete daily quizzes to progress
  • Explore elements through different learning modes
  • Take the final challenge on Day 7

Learning Modes

By Group: Learn elements organized by their chemical families (alkali metals, noble gases, etc.)
By Block: Study elements based on their electron configuration blocks (s, p, d, f)
By Atomic Number: Explore elements in order of increasing atomic number

Quiz Types

  • Multiple Choice: Select the correct answer from four options
  • Drag & Drop: Drag elements to their correct positions
  • Matching: Match elements with their properties
  • Fill in the Blank: Complete missing information

Keyboard Shortcuts

Space - Start/pause quiz
Enter - Submit answer
H - Show hint
T - Toggle theme
